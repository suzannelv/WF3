Les transitions

il existe 2 façons d'animer en CSS, on utilise les transition ou keyframes.

les transitions sont plus facile à manier, elles nous permettent de passer d'un état à un autre. pour cela, nous avons besoin d'information:

  • Une valeur initiale
  • une valeur finale
  • Une durée
  • un évènement déclencheur
  • une ou plusieurs propriétés CSS
